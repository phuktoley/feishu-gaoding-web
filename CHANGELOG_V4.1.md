# v4.1 版本更新说明

- **版本号**: v4.1

## 主要更新

### 1. CSV 导出格式优化

将 CSV 导出格式修改为符合稿定设计批量套版的标准格式。

#### 修改前（旧格式）
```csv
序号,主标题,副标题,记录ID
"1","港大AI作弊凉了","倒查！警示所有留学生","recv7fFnW7XEAT"
```

#### 修改后（稿定设计格式）
```csv
页面,文本_1,文本_2
"1","港大AI作弊凉了","倒查！警示所有留学生"
```

### 2. 字段映射关系

| 稿定设计字段 | 飞书表格字段 | 说明 |
|-------------|-------------|------|
| 页面 | 序号 | 自动生成的序号 (1, 2, 3...) |
| 文本_1 | 主标题 | 封面主文案 |
| 文本_2 | 副标题 | 封面副文案 |

### 3. 界面更新

- 按钮文字从"导出 CSV"改为"导出稿定格式"
- 表格列标题更新为"页面"、"文本_1 (主标题)"、"文本_2 (副标题)"
- 移除了"记录 ID"列的显示
- 更新了使用说明，明确说明导出格式

### 4. 文件命名

导出的文件名从 `feishu_data_YYYY-MM-DD.csv` 改为 `稿定设计-数据上传_YYYY-MM-DD.csv`

## 修改的文件

- `client/src/pages/Export.tsx` - 导出页面组件

## 使用方法

1. 访问"飞书数据"页面
2. 点击"导出稿定格式"按钮
3. 下载的 CSV 文件可直接上传到稿定设计的批量套版功能

## 稿定设计格式说明

根据稿定设计官方要求：

1. 每一行表格的内容将填充成一份设计结果
2. 请不要增加、删除、修改表头内容
3. 请不要合并、拆分单元格
4. 当前批量套版仅支持最多 148 行数据
