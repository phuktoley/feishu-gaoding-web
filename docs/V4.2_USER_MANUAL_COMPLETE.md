# V4.2 完整用户手册

## 概述

本手册介绍两个协同工作的应用：
1. **0103XHS** - 小红书爆款内容生成助手
2. **feishu-gaoding-web** - 飞书-稿定设计自动化工具

两个应用共同构成完整的小红书内容生产工作流。

## 应用访问地址

| 应用 | 地址 | 用途 |
|------|------|------|
| 0103XHS | https://0103xhs-production.up.railway.app | 生成文字内容 |
| feishu-gaoding-web | https://feishu-gaoding-web-production.up.railway.app | 生成封面图片 |

## 第一部分：0103XHS 使用指南

### 1.1 内容生成工作台

#### 单篇生成

1. **选择配置**
   - 业务场景：休学、延毕、学术不端、挂科、留学、退学
   - 目标情绪：共情型、警示型、求助型、成功型、吐槽型
   - 人设类型：过来人学姐、专业顾问、留学博主、学术达人
   - 留学地区（可选）：美国、英国、澳洲、加拿大等

2. **生成标题**
   - 点击"开始生成标题"
   - 系统生成 8 个标题选项，每个带评分（1-10分）
   - 选择满意的标题

3. **生成正文**
   - 点击"使用此标题生成正文"
   - 系统生成 300-500 字的正文
   - 正文符合选定的人设风格

4. **生成标签**
   - 点击"生成话题标签"
   - 系统生成 8 个相关话题标签

5. **生成封面**
   - 点击"生成封面文案"
   - 系统生成封面主文案和配色方案

#### 批量生成

1. 切换到"批量生成"标签
2. 设置生成数量（1-10篇）
3. 选择配置参数
4. 点击"批量生成 N 篇内容"
5. 等待生成完成（显示进度条）
6. 选择要导出的内容
7. 点击"导出到飞书多维表格"

### 1.2 人设管理

1. 点击侧边栏"人设管理"
2. 查看预设人设：过来人学姐、专业顾问、留学博主、学术达人
3. 可以编辑人设的：
   - 名称和描述
   - 写作风格
   - 常用emoji
   - 口头禅

### 1.3 历史记录

1. 点击侧边栏"历史记录"
2. 查看所有生成过的内容
3. 支持收藏、复制、重新导出

### 1.4 飞书导出配置

1. 点击"飞书配置"按钮
2. 填写配置信息：
   - **App ID**: 从飞书开放平台获取
   - **App Secret**: 应用密钥
   - **App Token**: 多维表格 URL 中的 token
   - **Table ID**: 多维表格 URL 中的 table 参数
3. 点击"保存配置"

## 第二部分：feishu-gaoding-web 使用指南

### 2.1 飞书配置

1. 访问"飞书配置"页面
2. 填写飞书应用凭证：
   - App ID
   - App Secret
3. 填写多维表格信息：
   - App Token（从 URL 获取）
   - Table ID（从 URL 获取）
   - 图片字段名称（默认"封面图片"）
4. 点击"测试连接"验证配置
5. 点击"保存配置"

### 2.2 获取飞书数据

1. 访问"飞书数据"页面
2. 系统自动加载多维表格数据
3. 查看数据预览（页面、主标题、副标题）
4. 点击"导出稿定格式"下载 ZIP 文件

### 2.3 在稿定设计生成图片

1. 访问 [稿定设计](https://www.gaoding.com)
2. 选择"批量套版"功能
3. 上传从本应用导出的 XLSX 文件
4. 选择模板并生成图片
5. 下载生成的 ZIP 文件

### 2.4 上传图片到飞书

1. 访问"上传图片"页面
2. 将稿定设计下载的 ZIP 文件拖拽到上传区域
3. 系统自动解析 ZIP 文件
4. 预览图片和对应的飞书记录
5. 点击"上传到飞书"
6. 系统自动将图片回填到飞书多维表格

## 第四部分：常见问题

### Q1: 飞书连接失败怎么办？

1. 检查 App ID 和 App Secret 是否正确
2. 确认应用已获得多维表格读写权限
3. 检查 App Token 和 Table ID 是否从正确的 URL 获取

### Q2: 生成的内容质量不满意？

1. 尝试更换人设类型
2. 在"补充信息"中添加更多背景
3. 点击"重新生成"获取新的结果

### Q3: 批量生成速度慢？

批量生成需要调用 AI 接口，每篇内容约需 30-40 秒。3篇内容约需 2 分钟。

### Q4: 图片上传失败？

1. 确认 ZIP 文件格式正确
2. 检查飞书表格的"封面图片"字段是否为附件类型
3. 确认应用有上传附件的权限
